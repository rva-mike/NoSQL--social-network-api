# NoSQL Social Network Api

  ## Description
   This is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It uses Express.js for routing, a MongoDB database, and the Mongoose ODM. In addition to using the Express.js (https://expressjs.com/) and Mongoose (https://www.npmjs.com/package/mongoose) packages, the app uses the native JavaScript Date object to format timestamps.
